# RAG Chatbot

An intelligent chatbot powered by Retrieval-Augmented Generation (RAG) technology.

## Features
- FastAPI backend
- Multiple LLM support (Gemini, OpenAI, Cohere)
- Vector search with Qdrant
- Real-time chat functionality

## API Endpoints
- `/` - Root endpoint
- `/health` - Health check
- `/docs` - API documentation
- `/api/chat` - Chat endpoint

## Documentation
Visit `/docs` for complete API documentation.
